ALTIRE API

API Endpoint https://floating-badlands-37340.herokuapp.com/api

User Routes

POST
/signup
{ name: { type: String, required: true, }, email: { type: String, required: true, unique: true }, phone: { type: String, required: true, unique: true, }, password: { type: String, required: true }, role: { type: Number, required: true }, }

NOTE: Role is the role of the user. 0 is regular customer, 1 is clothier, 2 is tailor, 3 is Admin

POST
/login
Body { email: { type: String, }, password: { type: String, }, }
Response { success: true, token: "long string of data", role: "0, 1, 2, 0r 3", userId: "users id" }

NOTE: Role is the role of the user. 0 is regular customer, 1 is clothier, 2 is tailor, 3 is Admin

Get
/user
Headers authorization: "Bearer ${token}"
Response { _id, name, email, phone, profileImage, address, role }

NOTE: The profile picture response is returned if the user has a profile picture, it was not required at signup but the user can upload a profile picture to the edit user route
